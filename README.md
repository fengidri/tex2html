# tex 字符分组
* 0 转义符         \
* 1 组开始         {
* 2 组结束         }
* 3 数学环境       $
* 4 表格对齐       &
* 5 换行           return
* 6 参数           #
* 7 上标           ^
* 8 下标           \_
* 9 可忽略的字符   null
* 10 空格
* 11 字母          (A-Z 和 a-z)
* 12 其它字符
* 13 活动符        ~
* 14 注释符        %
* 15 无用符        delete
